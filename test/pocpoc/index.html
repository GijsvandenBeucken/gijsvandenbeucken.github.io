2. Linked data editor

In deze test halen we nog geen gegevens op vanuit bronnen zoals het persoonregsiter.
Dit simuleren we door zelf data in te voeren in je SolidPod.
In het onderstaande veld kun je jouw woonplaats invoeren.
De input wordt weggeschreven naar je SolidPod.

Kijk mee in je SolidPOD door in te loggen op PodPro.
Ga in PodPro in het linkerscherm vervolgens naar de map profile en klik op card.
Je ziet in het rechterscherm de inhoud van je profielkaart.
Op regel 17 of 18 schrijf jij zodadelijk je woonplaats als vcard:locality.
Dit is een manier om je woonplaats te beschrijven
